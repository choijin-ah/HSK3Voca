HSK 3급 600단어 암기용 주제 분류

단어를 사전순이 아니라 “시험에서 같이 나오는 장면” 기준으로 묶었습니다. 총 600개 단어.

브라우저 저장 로그인하면 서버 저장

외우는 순서

처음부터 600개를 똑같이 외우지 말고, 1순위 → 2순위 → 3순위로 압축해서 보세요. 시험 직전에는 1순위 묶음만 반복해도 문장 해석 속도가 확 올라갑니다.

1회독: 뜻만 3초 안에 말하기중국어를 보고 뜻이 바로 안 나오면 체크합니다. 손으로 쓰는 암기는 뒤로 미루세요.
2회독: 묶음 제목으로 떠올리기예: 饭馆을 보면 菜单, 点, 菜, 筷子까지 같이 떠올리는 방식입니다.
3회독: 예문틀에 넣기“我 + 동사 + 목적어”, “今天/明天 + 문장”, “因为…所以…” 같은 틀에 단어를 끼워 넣습니다.
시험 직전: 기능어 먼저不, 没, 了, 的, 得, 地, 因为, 所以, 但是, 如果 같은 단어는 문장 구조를 바꾸므로 매일 보세요.

암기 규칙 5개

  1. 단어 하나씩 외우지 말고 장면으로 외우세요. 饭馆, 菜单, 点, 菜, 筷子는 한 장면입니다.
  2. 반대말은 무조건 같이 외우세요. 大/小, 多/少, 高/低, 近/远, 快/慢, 新/旧.
  3. 동사는 예문으로 외우세요. 看=보다가 아니라 “我看电影 (wǒ kàn diànyǐng)”처럼 외우는 편이 빠릅니다.
  4. 양사는 명사와 붙여 외우세요. 一本书, 一件衣服, 一张票, 一辆车, 一双鞋.
  5. 모르는 단어만 체크해서 다시 보세요. 아는 단어까지 반복하면 시간이 부족합니다.

암기용으로 명백한 원본 오타 일부는 바로잡았습니다. 예: 城市 chéngshì, 马 말(동물), 月亮 달.
